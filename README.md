# Bin Schedule
Send email alerts for your bin and recycling pickup days. 

## Prerequisites

Either:
- Docker
- Docker Compose

Or:
- Node 18+

## Running application
### Docker
```
docker-compose build
docker-compose up
```

### Node
```
node app
```
Or:
```
nodemon app
```
